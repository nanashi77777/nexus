### 示例代码（小型Transformer）

如果你想快速上手，以下是一个简化的PyTorch Transformer实现，适合初学者：

```python
import torch
import torch.nn as nn

class SimpleTransformer(nn.Module):
    def __init__(self, vocab_size, d_model=64, nhead=4, num_layers=2):
        super().__init__()
        self.embedding = nn.Embedding(vocab_size, d_model)
        self.transformer = nn.Transformer(d_model, nhead, num_layers)
        self.fc = nn.Linear(d_model, vocab_size)
    
    def forward(self, src, tgt):
        src = self.embedding(src)
        tgt = self.embedding(tgt)
        output = self.transformer(src, tgt)
        return self.fc(output)

# 示例使用
model = SimpleTransformer(vocab_size=10000)
src = torch.randint(0, 10000, (10, 32))  # 模拟输入序列
tgt = torch.randint(0, 10000, (10, 32))  # 模拟目标序列
output = model(src, tgt)
print(output.shape)  # [序列长度, 批次大小, 词汇表大小]
```

**说明**：这是一个简化的Transformer，可在Colab上运行。后续可添加位置编码、掩码等。
