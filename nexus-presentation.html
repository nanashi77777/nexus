技术选型 - 为什么选择这些技术

后端核心

  • Java 21 + Spring Boot 3：企业级稳定性与性能
  • 六边形架构：业务与技术彻底解耦

数据存储

  • PostgreSQL + PGVector：关系型与向量数据库一体化
  • Redis：高性能缓存与分布式锁

AI集成

  • LangChain4j：统一的大模型交互接口
  • Apache Tika：多格式文档解析

安全框架

  • Sa-Token：轻量级权限认证
  • BCrypt + AES：密码与数据加密

前端

  • Vue 3 + TypeScript：现代化前端开发体验

构建工具

  • Gradle (Kotlin DSL)：高效构建
